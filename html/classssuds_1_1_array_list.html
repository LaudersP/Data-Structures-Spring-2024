Detailed Description

ArrayList class for dynamic array of strings.

This class provides a basic functionality for managing an array of strings with dynamic memory allocation. This includes Appending, Inserting, Clearing, Getting, Finding, Removing, and Resetting of list strings. Along with capabilities to get the length of the list at its current state.
